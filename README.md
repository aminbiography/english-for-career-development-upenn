# English for Career Development – University of Pennsylvania

A structured single-page HTML/CSS application summarizing the core concepts of the *English for Career Development* course by the University of Pennsylvania, delivered via Coursera.

---

## Overview

This project presents a concise, visually structured summary of the course, covering the complete job application process. It combines academic content with a clean and modern UI layout suitable for portfolio presentation and documentation.

The application is designed with a focus on clarity, readability, and structured information delivery.

---

## Course Scope

The summary covers five core areas:

1. Entering the Job Market  
2. Résumé Development  
3. Cover Letter Writing  
4. Networking Skills  
5. Interview Preparation  

Each section integrates:

- Practical career development skills  
- Structured English communication  
- Real-world professional application  

---

## Features

- Single-file HTML and CSS implementation  
- Responsive layout with modern UI structure  
- Clear content hierarchy (hero, timeline, units, reflection)  
- Semantic and accessible HTML structure  
- Portfolio-ready design
